## A Dor

Todo social media de agência conhece essa rotina: o cliente manda um briefing vago pelo WhatsApp — *"quero algo sobre skincare, tipo aqueles vídeos que bombam"* — e começa a corrida. Você abre o Instagram, rola o feed por 40 minutos, salva uns posts, abre o TikTok, mais 30 minutos, tenta lembrar por que salvou aquele Reel, monta uma planilha improvisada, e quando finalmente senta pra escrever... já perdeu 2 horas só na pesquisa.

Multiplica isso por 5 clientes, 4 pautas por semana, e o tempo criativo vira tempo operacional. **O gargalo não é criar conteúdo — é encontrar a referência certa e traduzir o briefing do cliente em algo acionável.**

---

## O Insight

Em conversa com a Laura — profissional de agência de marketing para médicos — ficou claro que o processo de pesquisa + ideação + copywriting é o verdadeiro vilão. O cliente precisa ver referências para entender o que produzir. O social media precisa de dados reais de engajamento para justificar suas escolhas. E no final, o copy precisa sair pronto, adaptado pra plataforma, com gancho, CTA e hashtags.

Não faltam ferramentas de IA para escrever texto. **O que falta é uma que entenda o fluxo real de trabalho da agência: do briefing bagunçado ao conteúdo publicável.**

---

## A Solução

**Musa** é um workspace de conteúdo com inteligência artificial que comprime o ciclo de produção de horas para minutos. O coração do Musa é um pipeline de 4 etapas que transforma qualquer tema em copy pronto para postar:

### Dois modos de entrada

- **Busca Rápida** — Digite um tema, escolha plataforma (Instagram/TikTok) e formato (Reels/Carrossel/Post/Stories). Direto ao ponto.
- **Decodificador de Briefing** — Cole o e-mail, WhatsApp ou documento do cliente. A IA extrai tema, tom, público-alvo, plataforma e formato automaticamente, identifica lacunas e gera perguntas de clarificação.

## Como Funciona

O Musa vai além do pipeline principal. É um **workspace completo** com 8 ferramentas integradas:

### Musa Pipeline `/musa`
O coração do app. Busca referências virais reais via scraping, analisa com IA em 7 dimensões (tipo de hook, gatilho emocional, formato, arco narrativo, qualidade visual, potencial de comunidade, timing), gera 5 ideias criativas com níveis de dificuldade, e produz copy final com caption, hashtags, CTA e roteiro cena-a-cena.

### Copy Lab `/copy-lab`
Gerador de copy standalone — para quando você já sabe o que quer escrever. Escolha tipo (Ad, Social, Email, Headline, Carrossel), tom de voz, plataforma, e gere até 5 variações de uma vez. Integra brand voice do cliente selecionado.

### Central de Briefs `/briefs`
Decodificador de briefings com 4 templates (Campanha, Lançamento, Sazonal, Livre). Cole o briefing bagunçado → receba campos estruturados + perguntas de clarificação. Envie direto para o Musa Pipeline ou Copy Lab com um clique. Histórico completo de briefs decodificados.

### Hub de Clientes `/clients`
Gerencie perfis de clientes com segmento, brand voice, público-alvo, plataformas e formatos preferidos. O brand voice é injetado automaticamente em todas as gerações de copy quando o cliente está selecionado. Página de detalhe com métricas, copies recentes e briefs associados.

### Carrossel Visual `/carousel`
Construtor de carrossel com preview ao vivo. Gere slides com IA (Cover, Content, Statistic, Quote, List, CTA), personalize cores, layout e tipografia. Drag-and-drop para reordenar. Integração direta com o pipeline — após gerar copy no Musa, clique "Criar Carrossel Visual" para montar os slides automaticamente.

### Calendário de Conteúdo `/calendar`
Pipeline visual de publicação. Crie entradas com cliente, plataforma, formato e status (Rascunho → Agendado → Publicado). Arraste para reagendar. Filtros por cliente, status e plataforma. Visão mensal com indicador de hoje.

### Relatórios `/reports`
Cole métricas de qualquer fonte (impressões, alcance, engajamento, seguidores) e a IA gera um relatório estruturado com análise, destaques, recomendações acionáveis e conclusão.

### Brand Book `/brand`
Identidade visual e guidelines do Musa. Design tokens, CSS e referência rápida disponíveis para download.

## Tech Stack

| Camada | Tecnologia | Justificativa |
|--------|-----------|---------------|
| **Framework** | Next.js 15 (App Router) | Server Components, API Routes, edge-ready |
| **Linguagem** | TypeScript | Type safety em todo o codebase |
| **Styling** | Tailwind CSS 4 | Glassmorphism, aurora effects, CSS-only animations GPU-composited |
| **UI Components** | shadcn/ui + Radix | Acessíveis, composáveis, customizáveis |
| **Animações** | Framer Motion + CSS Keyframes | Transições suaves, aurora background em `translate3d` |
| **IA** | Claude Sonnet 4 (Anthropic) | Análise, ideação e copywriting de alta qualidade |
| **Scraping** | Apify Actors | Instagram & TikTok em tempo real, fallback automático |
| **Banco de Dados** | Supabase (PostgreSQL) | Clientes, briefs, histórico de copy, atividades, calendário |
| **Deploy** | Vercel | Edge-optimized, zero config, preview deploys |

### Arquitetura

```
Browser ──→ Next.js App Router ──→ API Routes ──→ Claude API (Anthropic)
                                              ──→ Apify (Scraping)
                                              ──→ Supabase (Persistência)

State: React Context (workspace) + localStorage (cache) + Supabase (source of truth)
```

11 API routes servem o app: `/api/search`, `/api/analyze`, `/api/ideas`, `/api/copy`, `/api/brief`, `/api/copylab`, `/api/carousel`, `/api/report`, `/api/clients`, `/api/calendar`, `/api/activities`.
