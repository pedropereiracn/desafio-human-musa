01
Brand Story & Mission
A origem da Musa e o propósito que guia cada decisão de produto.

"Na mitologia grega, as Musas eram as deusas que inspiravam a criação artística. Musa nasce dessa mesma essência — ser a fonte de inspiração que transforma dados brutos em conteúdo que conecta, engaja e converte."

Missão

Empoderar social media managers com inteligência de conteúdo alimentada por IA, eliminando o trabalho repetitivo e liberando tempo para criatividade estratégica.

Visão

Ser a plataforma definitiva que conecta criatividade humana com análise de IA, transformando a forma como agências produzem conteúdo para redes sociais.

Nossos Valores

🎯

Precisão

Dados acurados, sugestões relevantes. Cada insight é verificado e cada recomendação é baseada em evidências.

⚡

Velocidade

De briefing a copy pronta em minutos, não horas. Workflow otimizado para a realidade de agências.

✨

Criatividade

IA como amplificador criativo, nunca como substituto. Potencializamos a voz humana com inteligência artificial.

🌐

Acessibilidade

Tecnologia avançada com interface intuitiva. Qualquer social media manager consegue usar desde o primeiro dia.

Focus Ring

Todos os elementos interativos devem ter um focus ring visivel ao navegar com teclado.

:focus-visible {
outline: 2px solid #22C55E;
outline-offset: 2px;
}

Navegacao por Teclado

Tab: Avanca entre elementos interativos
Shift+Tab: Retrocede entre elementos
Enter/Space: Ativa botoes e links
Escape: Fecha modals e dropdowns
Arrow keys: Navega dentro de menus e tabs

Next.js Integration

// app/layout.tsx
import { Space_Grotesk, Inter } from 'next/font/google'
import localFont from 'next/font/local'

const spaceGrotesk = Space_Grotesk({
  subsets: ['latin'],
  variable: '--font-display',
  weight: ['400', '500', '600', '700'],
})

const inter = Inter({
  subsets: ['latin'],
  variable: '--font-body',
})

export default function RootLayout({ children }) {
  return (
    <html className={`${spaceGrotesk.variable} ${inter.variable}`}>
      <body className="bg-surface-0 text-neutral-50 font-body">
        {children}
      </body>
    </html>
  )
}
